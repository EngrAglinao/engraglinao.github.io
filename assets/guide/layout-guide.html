Mobile Input Rules (add to your CSS)
  • • Minimum touch target height: 44px on all mobile sizes
  • • Set font-size: 16px on all inputs for iOS — prevents auto-zoom on focus
  • • Use type="email" and type="tel" to trigger correct mobile keyboards
  • • Make all inputs width: 100% on XS/SM — already true with Bootstrap .form-control
  • • "Send Message" button should be width: 100% on mobile
